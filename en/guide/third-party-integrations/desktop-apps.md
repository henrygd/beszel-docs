# Desktop Applications

Unofficial desktop clients for Beszel are maintained by third parties. You need a working Beszel instance accessible from your device to use them.

## BeszelBar (macOS)

**BeszelBar** is an unofficial native macOS menu bar client for Beszel.

- **GitHub:** [Loriage/BeszelBar](https://github.com/Loriage/BeszelBar)

### Features

- **System overview** — All servers at a glance with status, CPU, memory, and disk usage.
- **System details** — Hover for temperatures, uptime, and detailed usage bars.
- **Container monitoring** — Docker containers with health status, resource usage, and image info.
- **Alerts** — Active alerts with triggering conditions, reflected on the menu bar icon.
- **Multi-hub support** — Manage multiple Beszel instances (e.g. work, home, personal) and switch between them.
- **Auto-refresh** — Configurable polling interval from 10 seconds to 5 minutes.
- **Polished experience** — Native SwiftUI + AppKit, Keychain credential storage, signed and notarized.

### Installation

- Install via Homebrew: `brew install --cask loriage/tap/beszelbar`, or
- Download the latest release from [Releases](https://github.com/Loriage/BeszelBar/releases).
